简介：
本项目展示基于我在中信银行国际（香港）2023年暑期实习的实际工作流程重建，使用python绘制桑基图分析了app内点击流，本项目使用合成数据重建流程，所有数据均为模拟生成
仓库内容：
simulatedData.csv-合成的模拟原始数据
monitorClickData.ipynb - 生成模拟数据的代码
cleanedSimulatedData.csv - 清理后的数据
cleanSimulatedData.ipynb - 数据清理代码
analysisSimulatedData.ipynb - 绘图与分析所用的代码，包含若干未纳入summary的探索性图表
summary.pdf - 一页总结（A4），总结了分析的可交付成果
